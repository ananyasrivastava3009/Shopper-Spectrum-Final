# Shopper Spectrum – Customer Segmentation System

## Project Overview
This project performs customer segmentation using Machine Learning (K-Means Clustering) and provides a Streamlit web application for customer analysis.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Features
- Customer Segmentation
- K-Means Clustering
- Interactive Streamlit Dashboard
- Customer Insights

## How to Run
Run project.ipynb to generate models/similarity_df.pkl before running the Streamlit app.
```bash
pip install -r requirements.txt
streamlit run streamlit_app/app.py
